MMM Extensions
==============

.. module:: mmm_framework.mmm_extensions

The mmm_extensions module provides advanced modeling capabilities including
nested/mediated models, multivariate outcomes, and variable selection.

.. note::

   This module uses lazy loading for heavy PyMC dependencies. Import the
   specific classes you need rather than using ``from mmm_extensions import *``.
