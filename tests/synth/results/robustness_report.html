1The test data is not the model's generative process

The model's existing fixtures draw data from (essentially) the same family it assumes, so a passing test only proves the sampler works when reality matches the model. Every panel below shows a test dataset that deliberately deviates from a model assumption. The one dataset that does match — clean — is the positive control, recovered to 7% with 100% coverage.

Saturation shape — the model can't bend like the data

model assumes Model assumes a strictly concave 1−exp(−λx) curve (diminishing returns from the first dollar).

the data does saturation_misspec data is S-shaped (Hill, coef 3): a low-spend threshold, then a steep rise.

→ 46% median error, 25% coverage — and PPC + refutation both pass. Silent.

Carryover shape & window — truncated and mis-peaked

model assumes Model uses geometric decay truncated at l_max = 8 weeks, peaking at lag 0.

the data does adstock_misspec data is a delayed Weibull peaking ~6–8 weeks out with mass past 26 weeks.

→ 77% median error — caught here only by the robustness value (Search flagged fragile).

Unobserved confounding — spend is not exogenous

model assumes Model assumes media spend is exogenous (no hidden driver of both spend and sales).

the data does Search spend chases a hidden demand signal (corr 0.35); that signal also lifts sales.

→ 41% median error; Search over-credited +153%; truth outside the CI. Silent.

Multicollinearity — channels rise and fall together

model assumes Model assumes enough independent spend variation to separate each channel's effect.

the data does All four channels share one flighting calendar (mean pairwise corr 1.00).

→ 29% median error, total media right (−2%) but the per-channel split is scrambled. Silent.

Spend outlier — one spike compresses the normalizer

model assumes Model normalizes each channel by its training max, assuming the max is representative.

the data does A 15× data-entry spike sets the max; every real week collapses toward 0 on the curve.

→ 46% median error, 0% coverage — no channel's truth lands in its interval. Silent.

Time-varying effectiveness — not one constant β

model assumes Model fits one constant coefficient per channel for the whole window.

the data does TV fatigues over two years; Search jumps at a mid-series break (algo/creative change).

→ 11% median error — the window total absorbs the drift (a time-resolved estimand would not).

3Convergence

National single-KPI fits converge cleanly — r̂ ≈ 1.005–1.01, 0 divergences everywhere except the Aurora kitchen-sink (extension model). But the production gate (r̂ < 1.01, strict) false-alarms because an inherent β·λ identifiability ridge parks r̂ right at 1.01.

Why r̂ sticks at ~1.01. A channel's contribution is ≈ β·λ·x in the linear regime, so only the product β·λ is identified — a curved ridge a diagonal NUTS mass matrix can't rotate onto. The lowest-ESS parameters are always the media/saturation terms: beta_Search (1427), sat_lam_Search (1447), beta_TV (1795), channel_contributions (1799). More draws help (300→1200 lifts min-ESS 790→3,112).
Where divergences appear. Collinear spend, the legacy 2-point adstock blend, single-group hierarchies, and Hill + latent mediation (Aurora: r̂ 1.11, 108 divergences).

4Fit-time & scaling

A 156-week / 4-channel fit is fast; the analyses around it dominate. numpyro is ~2× faster than pyMC; channels scale super-linearly; the refutation suite costs more than the fit itself.

Pure fit() wall-clock by config

One knob varied at a time from the baseline. numpyro halves the time; 8 channels is ~2.7× the 4-channel fit.

Cost of a scored scenario

The refutation suite (4 refits) is the cost center — bigger than the fit. Contributions are O(channels) predictive passes.

Scaling laws. Time ∝ n_obs (≈ weeks × geos) and ∝ draws; super-linear in channels (parametric adstock runs an O(n_obs·l_max) convolution inside every leapfrog step, per channel). At geo scale the obs-sized deterministics in the trace can reach multiple GB — compute the decomposition post-hoc instead of storing it every draw.

D. Statistical & operational

high confirmed
adstock·saturation·β equifinality
Many (decay, λ, β) triples fit equally — non-identified even with infinite data.
medium confirmed
Prior–data conflict (downward shrink)
Gamma(μ=1.5) β prior pulls large true effects down — the clean control's ~7% under-recovery.
medium latent (verified)
Counterfactual HDI inflation
Default random_seed=None unpairs noise → CI ~√2 too wide. The harness passes a seed, so its coverage is honest.
medium confirmed
Strict r̂<1.01 gate false-alarms
The ridge parks r̂ at ~1.01 → fast fits flagged 'not converged' despite 0 divergences.
The one real fix. The ridge, collinearity, and confounding are identification problems no sampler setting solves. Anchoring effects with randomized geo-lift / incrementality experiments (calibration/) pins β, collapses the ridge, and is the one signal these silent failures cannot fake.
