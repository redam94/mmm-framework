Methodology

Model Specification

This analysis uses a Bayesian Marketing Mix Model with the following components:

  • Likelihood: Normal with estimated scale
  • Baseline: linear trend + Fourier seasonality
  • Media effects: Hill saturation × Geometric adstock
  • Controls: As specified
  • Priors: Weakly informative

Inference via MCMC (4 chains, 2000 samples each, 1000 warmup).
