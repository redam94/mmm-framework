# Competitive Analysis — MMM Platforms

A fact-checked comparison of this MMM Framework against the platforms marketers actually shortlist.
Built 2026-07-08 via a multi-agent research workflow: the app was characterized directly from its
source tree (shipped vs. deferred, not marketing copy); each competitor was researched from primary
sources and then **adversarially fact-checked**, with corrections logged and a confidence grade assigned.

**Snapshot:** mid-2026 · 9 platforms · 15 dimensions · ~90 primary sources.

## Files

| File | What it is |
| --- | --- |
| [`competitor-analysis.md`](./competitor-analysis.md) | The reference writeup — positioning, the full 15×9 matrix, per-dimension detail, all nine platform profiles, wins/lags, a "choose by use case" guide, and the bottom line. Diffable and editable. |
| [`mmm-competitor-comparison.html`](./mmm-competitor-comparison.html) | The interactive version — a self-contained page (open in any browser) with a scannable rating matrix, expandable detail, collapsible profiles, light/dark themes. Also published as a shareable Artifact. |
| [`research-data.json`](./research-data.json) | The raw verified data behind both — every profile with its **source URLs**, confidence grade, and the specific corrections the verification pass applied. Regenerate `competitor-analysis.md` from this. |

## Platforms covered

This app · Meta Robyn · Google Meridian · PyMC-Marketing · Recast · Mutinex GrowthOS ·
Google LightweightMMM (legacy) · Analytic Partners · the enterprise/consulting category
(Ipsos MMA, Kantar, Circana/ex-Nielsen, Keen).

## Caveats

- Competitor head-to-head benchmarks (e.g. PyMC Labs vs. Meridian, Recast vs. Robyn) are
  **competitor-sourced** and treated as directional, not settled fact.
- Pricing for closed vendors is third-party estimate — none publish price lists.
- Point-in-time snapshot; open-source versions, funding, and analyst positions drift.
- Independent analysis; not affiliated with, endorsed by, or reviewed by any vendor named.
